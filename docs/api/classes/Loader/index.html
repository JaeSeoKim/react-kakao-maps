Class: Loader

Kakao Map Api Loader

new Loader(options).load() 함수를 이용하여 Api를 비동기적으로 삽입할 수 있습니다.

해당 Loader를 이용시 react-kakao-maps-sdk 내부에서 injection 되는 이벤트를 감지하여 kakao map api 로딩 이후에 렌더링을 진행합니다.
