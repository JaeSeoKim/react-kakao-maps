## Usage

이 예제를 실행하기 위해서는 **필수적**으로 Kakao Map App Key를 환경변수로 추가해야 합니다.

```shell
# /examples/nextjs-app/.env
NEXT_PUBLIC_APPKEY=발급받은APPKEY
```

### 실행하기

```shell
pnpm install
pnpm run build:pacakges
pnpm run build:examples
```
